# **App Name**: MediVue

## Core Features:

- Real-time Vital Signs Display: Display real-time vital signs (BP, pulse, temperature, SpO2, respiratory rate) for each patient.
- Movement/Sleep Pattern Visualization: Visualize patient movement and sleep patterns to identify potential issues.
- Bed Status Monitoring: Monitor bed position and adjustment status in real time.
- Medication Schedule & Reminders: Display medication schedules and send reminders to patients and staff.
- Priority-Based Alert System: Implement a priority-based alert system (Emergency/High/Medium/Low) for critical events.
- Fall Detection Alerts: Provide immediate alerts upon fall detection, enhancing patient safety.
- Preventive Alerts Tool: Employ a generative AI tool to determine the likelihood of pressure sores. If high risk, issue preventive alerts based on the data and reasoning.
- Food/Water/Medicine Reminders: Reminder system for food, water, and medicine intake.

## Style Guidelines:

- Primary color: Calm and professional light blue (#A0CFEC).
- Background color: Light gray (#F0F4F7) for a clean, modern look.
- Accent color: Muted teal (#70A1AF) for key actions and highlights.
- Body and headline font: 'Inter', a sans-serif font for a modern, objective feel.
- Use clear, recognizable icons for vital signs and alerts.
- Maintain a clean, intuitive layout for easy navigation and data interpretation.
- Subtle transitions and animations for a smooth, responsive user experience.
